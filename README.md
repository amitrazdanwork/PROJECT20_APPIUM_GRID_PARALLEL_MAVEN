**PROJECT - APPIUM_GRID_MAVEN**


Mobile test automation framework for performing mobile testing on Application under test.
This project involves usage of Appium along with Selenium grid setup (Hub and nodes) for parallel execution of tests where all nodes is treated as an emulator environment.

**TECH STACK USED: -**


**Programming language:** Java

**Controlling test execution:** testNG library

**Report generation:** Extent reporting

**Parallel execution:** Selenium Grid (Distributed setup in form of Hub and node(s) ) + Maven plugin (surefire plugin)

**Mobile app automation:** Appium
